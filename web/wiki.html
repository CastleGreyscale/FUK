Overview

FUK (Framework for Unified Kreation) is a local-first AI rendering pipeline built for VFX professionals. It provides a structured, stable environment for image and video generation that integrates cleanly into existing production pipelines — without telemetry, cloud dependencies, or workflows designed around consumer use cases.

FUK runs entirely on your hardware. Models, outputs, and project data stay on your machine.

FUK is in active development. Core generation is stable. AOV outputs are functional but being refined. See the roadmap section on the main page for current focus areas.

Installation

Prerequisites

  • Python 3.10+
  • CUDA 11.8+ (NVIDIA) or compatible AMD drivers
  • Git
  • FFmpeg

Steps

  1. Clone the repository:
    git clone https://github.com/CastleGreyscale/FUK
  2. Install dependencies:
    pip install e .
  3. Download required models — see File Locations for expected paths
  4. Run Setup:
    chmod +x setup.sh
  5. Choose models - edit config/models.json - remove unwanted models
  6. Set model paths - edit config/defaults.json - ret model root directory
  7. Download models:
    ./download_models.sh
  8. Launch FUK:
    ./start.sh

See the GitHub README for the complete and up-to-date installation instructions.

Preprocessors

Preprocessors convert input images into conditioning signals that guide generation — giving precise control over pose, depth, structure, and edges in the output without fighting the model through prompt engineering.

OpenPose

Detects human body pose from an input image and extracts a skeleton map. Use this to replicate or constrain body positions in generated output.

  • Full body keypoint detection
  • Hand and face keypoints [if supported]

Depth

Generates a depth map from an input image, encoding near-to-far distance as a greyscale gradient. Used for spatial conditioning — maintaining scene depth structure across generations.

Edge Detection

Extracts edges and structural lines from an input image. Useful for preserving architectural forms, hard-surface geometry, or any scene with defined edges that should carry through to the output.

Using Preprocessors

  1. Select a preprocessor in the interface
  2. Load or specify an input image
  3. Run the preprocessor — the output is used automatically as a ControlNet conditioning input
  4. Adjust conditioning weight to balance prompt influence vs. structural control

Regional Conditioning (EliGen)

EliGen integration brings regional conditioning directly into the generation pipeline. Define compositional zones within the frame and assign independent parameters to each — separate prompts, weights, and conditioning per region.

How It Works

Rather than fighting the model for spatial control through prompt engineering alone, EliGen lets you explicitly partition the frame into regions. Each region gets its own prompt and conditioning weight, giving you direct control over composition.

Frame Interpolation

FILM (Frame Interpolation for Large Motion) synthesizes intermediate frames to increase the effective frame rate of generated video. Useful for smoothing motion or converting a lower frame count output to a deliverable rate.

Usage

  1. Generate or import a video sequence
  2. Select an interpolation factor (2×, 4×, etc.)
  3. Run interpolation — output is a new sequence at the higher frame rate

Parameters

ParameterDescription
Interpolation FactorFrame rate multiplier (2× doubles frames, 4× quadruples)

AOVs & Passes

FUK generates Arbitrary Output Variables (AOVs) alongside the beauty render. These technical passes enable compositing, grading, and integration with professional DCC tools downstream.

AOV outputs are currently being refined. Quality and coverage are improving. Feedback on specific issues is appreciated — contributions welcome.

EXR Export

FUK exports multi-layer 32-bit EXR files containing all available AOV passes in a single file. EXR is the standard interchange format in professional VFX and compositing pipelines.

Layer Structure

LayerChannelsDescription
RGBAR, G, B, ABeauty render
depthZDepth pass (32-bit float)
normalR, G, BNormal pass
crypto_*R, G, B, ACryptomatte layers

Compatible Applications

  • Blender Compositor
  • Houdini

History Panel

Features

  • Browsable history of all generations organized by project hierarchy
  • Per-entry metadata display: seed, parameters, model, timestamp
  • Re-queue any historical generation with identical parameters
