Built for Production

Professional AI generation tools shouldn't force you to work around them. FUK integrates into your existing workflows with proper project organization, native lossless EXR support, and stable infrastructure that doesn't break with every update.

Your hardware. Your models. Your data. No mandatory cloud services. No compromises.

System Requirements

What you'll need to run FUK effectively on your local hardware.

Storage Space

  • Base Install: ~15GB
  • Models: ~500GB (depending on selection)

Hardware (Tested On)

  • NVIDIA RTX 4090 (24GB VRAM)
  • AMD Threadripper
  • 256GB RAM
  • M2 SSDs

Operating Systems

  • Linux: Primary platform (Ubuntu 22.04+)
  • Windows: WSL2 recommended

Dependencies

  • Python 3.10+
  • CUDA 11.8+ (for NVIDIA)
  • FFmpeg for video processing
  • See GitHub for complete list

Core Functionality

Professional Generation

Image and video generation with proper project/shot/version organization. Industry-standard hierarchies that integrate with existing pipelines.

Technical Outputs

Depth maps, normal passes, cryptomattes. Multi-layer 32-bit EXR export ready for professional applications.

Controlled Workflows

Preprocessor integration for OpenPose, depth, edge detection and regional conditioning via masked zones. Generate with precision using external image inputs.

Post-Processing

Upscaling via Real-ESRGAN and frame interpolation via FILM. Professional quality enhancement for generated content.

Local Control

Runs entirely on your hardware. No telemetry, no cloud requirements, no surprise authentication. Your data stays yours.

Stable Infrastructure

Built on stable APIs without fragile dependencies. Tools that work today will work tomorrow without breaking your pipeline.
